# ERP System Frontend

A modern, comprehensive Enterprise Resource Planning (ERP) system built with React, TypeScript, and Vite. This application provides a complete solution for business management with an intuitive and responsive user interface.

## 🚀 Features

- **Modern UI/UX**: Built with Radix UI components and Tailwind CSS for a polished, professional interface
- **TypeScript**: Fully typed codebase for enhanced developer experience and reduced bugs
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modular Architecture**: Well-organized component structure for maintainability
- **Form Management**: Integrated with React Hook Form for efficient form handling
- **Data Visualization**: Recharts integration for comprehensive reporting and analytics
- **PDF Generation**: jsPDF integration for document generation
- **Internationalization**: Support for multiple languages with flag-icons
- **Theme Support**: Dark/light mode with next-themes

## 🛠️ Tech Stack

- **Framework**: React 18.3.1
- **Language**: TypeScript 5.6.3
- **Build Tool**: Vite 6.3.5
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **Icons**: Lucide React
- **Routing**: React Router DOM 7.12.0
- **Forms**: React Hook Form 7.55.0
- **Charts**: Recharts 2.15.2
- **PDF**: jsPDF 3.0.4
- **Date Picker**: react-day-picker 8.10.1

## 📋 Prerequisites

- Node.js 20.x or higher
- npm, yarn, or pnpm

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/Rami-Saab/ERPSystem-Frontend.git
cd ERPSystem-Frontend
```

2. Install dependencies:
```bash
npm install
```

## 🎯 Usage

### Development Server

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Production Build

Build for production:
```bash
npm run build
```

The optimized build will be in the `dist` directory.
